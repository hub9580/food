    포인트 적립/사용/소멸

    1. 결제금액의 3% 적립 (부가세 제외, 할인내역 제외, 무료 식사권, 쿠폰, 금액 할인권, 당사에서 지정한 제외 상품 등 적립 불가)
    2. 적립된 포인트는 3,000p부터 10p 단위로 사용 가능 (기타 할인 혜택과 중복 사용 가능하며, 결제 시 포인트 선 차감 후 타 할인이 적용)
    3. 적립된 포인트 사용 시, 포인트 사용 비밀번호 숫자 6자리 입력 후 사용 가능
    4. 포인트 사용 비밀번호 분실 시, 고객센터(1661-8514) 문의 후 포인트 사용 새 비밀번호 설정 가능
    5. 포인트는 포인트 적립 연도 기준 다음해 12월 31일 24시에 일괄 소멸

    기타 클럽 정보

    1. 매장에서 받은 클럽 카드는 홈페이지, 어플, 모바일 웹에서 등록 후 바로 사용 가능합니다.
    (단, 웰컴 쿠폰은 부메랑 카드 신규 등록 시부터 60일 기간 내에 1회 사용이 가능하며, 생일 쿠폰은 등록된 생일 일자에 맞춰 쿠폰이 탑재됩니다)
    2. 클럽 카드 분실 시, 가까운 매장에서 수령 후 홈페이지에 분실신고 처리 > 카드번호 재등록 후 사용 가능합니다.
    3. 한 번 결제 시, 하나의 카드만 사용 가능합니다. (예를 들어, A카드로 쿠폰을 쓰고, B카드로 적립하는 것은 불가)
    4. A카드로 쿠폰을 사용할 경우, A카드로만 적립 또는 할인 가능합니다. (B카드 적용 불가)
    5. 딜리버리 메뉴 및 서비스 이용 시, 모든 혜택(포인트 적립/할인/사용, 쿠폰 사용) 이용 불가합니다.

    카드사 혜택

    더욱 즐겁게 이용하는 방법! 매장에서 바로 현장 할인 후, 저렴하고 맛있게 즐겨보세요!

    할인 서비스 안내

    - 각 제휴카드 별로 할인가능 횟수, 전월 실적, 최대 할인금액 등의 상세 조건에 따라 할인이 제한될 수 있으니 자세한 안내는 카드사로 확인 부탁드립니다.
    - 모든 제휴카드는 중복 사용이 불가합니다.(단, 부메랑카드 할인은 적용 불가 / 부메랑 카드 적립은 타 제휴카드 할인과 중복 가능)
    - 모바일 카드 결제는 적용이 불가합니다.

    - 0.5%~2% 맴버십리워즈 적립
    - Be My Guest 서비스
